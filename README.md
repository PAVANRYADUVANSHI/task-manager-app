# Task Manager App ✅

Full-stack task management application with CRUD operations, user authentication, and real-time updates.

## 🚀 Features
- User authentication (Register / Login)
- Create, Read, Update, Delete tasks
- Task status tracking (Pending / In Progress / Done)
- Real-time UI updates
- Responsive design

## 🛠️ Tech Stack
- **Frontend:** React
- **Backend:** Node.js
- **Database:** SQL

## ⚡ Quick Start

### Backend
```bash
cd backend
npm install
npm start
```

### Frontend
```bash
cd frontend
npm install
npm start
```
